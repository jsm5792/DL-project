# DL-project

# 파일 설명

- `null_text_w_ptp_final_without_result.ipynb`: 출력 결과물이 모두 지워진 상태의 노트북 파일입니다. (실행용)
- `null_text_w_ptp_final_with_result.ipynb`: 이전 실행의 출력 결과물이 모두 남아있는 상태의 노트북 파일입니다. (참고용)

# 실행 가이드

정상적인 실행을 위해 다음 파일들을 작업 경로에 업로드한 후 노트북을 실행해 주세요.

1. 원본 이미지 파일
2. 합성하고자 하는 이미지 파일
3. `ptp_utils.py`
4. `seq_aligner.py`


## Files

* **`null_text_w_ptp_final_without_result.ipynb`**: Notebook with all outputs **cleared**. Use this to run the code from scratch.
* **`null_text_w_ptp_final_with_result.ipynb`**: Notebook with all outputs **preserved**. Use this as a reference to see the expected results.

## Usage

To run the notebook successfully, please upload the following files to your working directory:

1. Original Image (Source)
2. Image to Composite (Target)
3. `ptp_utils.py`
4. `seq_aligner.py`

Once uploaded, you can run the cells in order.
